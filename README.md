# GoodNews
GoodNews is a News App consuming the [NewsAPI](https://newsapi.org)

## Run Application
1. Open terminal 
2. Clone this repository\
   `git clone https://github.com/marianobattaglia/GoodNews.git`
3. Open project file GoodNews.xcodeproj
